# User Journey Maps

Visual representations of the steps users take to accomplish goals on the Nebari marketplace.

## Journey Map Legend

| Symbol | Meaning |
|--------|---------|
| **Touchpoint** | User interaction with platform |
| **Emotion** | User feeling at this stage |
| **Pain Point** | Friction or frustration |
| **Opportunity** | Potential improvement |

**Emotions Scale:**
- Delighted
- Satisfied
- Neutral
- Frustrated
- Angry

---

## Journey 1: First-Time Buyer

**Persona:** Thandi (Hobbyist) or Lerato (Casual Browser)
**Goal:** Find and purchase first bonsai

### Stages

#### 1. Awareness

| Aspect | Details |
|--------|---------|
| **Touchpoint** | Google search, social media, friend referral |
| **Actions** | Searches "buy bonsai South Africa", clicks ad/link |
| **Thoughts** | "I wonder if there's a good place to buy bonsai online" |
| **Emotion** | Curious |
| **Opportunities** | SEO optimization, social proof on landing page |

#### 2. Exploration

| Aspect | Details |
|--------|---------|
| **Touchpoint** | Homepage, Discover page |
| **Actions** | Browses listings, uses filters (price, location) |
| **Thoughts** | "Can I find something in my budget near me?" |
| **Emotion** | Hopeful → Satisfied (if good selection) |
| **Pain Points** | Overwhelmed by options, unsure what's "good" |
| **Opportunities** | Beginner-friendly filters, "Staff picks" section |

#### 3. Evaluation

| Aspect | Details |
|--------|---------|
| **Touchpoint** | Listing detail page, Seller profile |
| **Actions** | Reviews photos, checks seller ratings, reads reviews |
| **Thoughts** | "Is this tree healthy? Can I trust this seller?" |
| **Emotion** | Cautious |
| **Pain Points** | Photo quality varies, want more tree info |
| **Opportunities** | Care difficulty rating, health indicators |

#### 4. Contact

| Aspect | Details |
|--------|---------|
| **Touchpoint** | Messaging system |
| **Actions** | Sends message to seller with questions |
| **Thoughts** | "I hope they respond quickly" |
| **Emotion** | Anxious waiting |
| **Pain Points** | Slow seller responses |
| **Opportunities** | Response time indicators, notification push |

#### 5. Purchase Decision

| Aspect | Details |
|--------|---------|
| **Touchpoint** | Messages, possibly WhatsApp |
| **Actions** | Negotiates price, arranges payment/pickup |
| **Thoughts** | "This seems legit, I'll go for it" |
| **Emotion** | Excited |
| **Pain Points** | No in-app payment (trust concern) |
| **Opportunities** | Payment integration (future), escrow option |

#### 6. Completion

| Aspect | Details |
|--------|---------|
| **Touchpoint** | In-person pickup or delivery |
| **Actions** | Receives tree, inspects, pays |
| **Thoughts** | "It looks just like the photos!" |
| **Emotion** | Delighted (if good) / Frustrated (if not) |
| **Pain Points** | Tree condition differs from photos |
| **Opportunities** | Photo verification, dispute resolution |

#### 7. Post-Purchase

| Aspect | Details |
|--------|---------|
| **Touchpoint** | Rating system, Forum |
| **Actions** | Rates seller, asks care questions on forum |
| **Thoughts** | "I want to help others and learn more" |
| **Emotion** | Satisfied, Engaged |
| **Opportunities** | Prompt for review, suggest forum topics |

## Journey 2: Seller Listing a Bonsai

**Persona:** Johan (Collector) or Pieter (Nursery)
**Goal:** List and sell a bonsai from collection

### Stages

#### 1. Motivation

| Aspect | Details |
|--------|---------|
| **Trigger** | Collection thinning, need space/funds, tree outgrew interest |
| **Thoughts** | "Where can I reach serious buyers?" |
| **Emotion** | Determined |

#### 2. Account Setup

| Aspect | Details |
|--------|---------|
| **Touchpoint** | Signup, Profile setup |
| **Actions** | Creates account, fills profile, uploads avatar |
| **Thoughts** | "This is easy enough" |
| **Emotion** | Neutral → Satisfied |
| **Pain Points** | Long form, unclear what's required |
| **Opportunities** | Progressive profile completion |

#### 3. Listing Creation

| Aspect | Details |
|--------|---------|
| **Touchpoint** | Seller Dashboard, New Listing dialog |
| **Actions** | Uploads photos, fills details, sets price |
| **Thoughts** | "I want my tree to look good" |
| **Emotion** | Invested |
| **Pain Points** | Photo upload limits, unsure of pricing |
| **Opportunities** | Price suggestions, photo tips |

#### 4. Waiting for Interest

| Aspect | Details |
|--------|---------|
| **Touchpoint** | Dashboard stats, Notifications |
| **Actions** | Checks view count, waits for messages |
| **Thoughts** | "Is anyone seeing my listing?" |
| **Emotion** | Anxious |
| **Pain Points** | Low visibility, no feedback |
| **Opportunities** | Listing optimization tips, boost option |

#### 5. Buyer Inquiry

| Aspect | Details |
|--------|---------|
| **Touchpoint** | Messages |
| **Actions** | Responds to questions, negotiates |
| **Thoughts** | "This buyer seems serious" |
| **Emotion** | Hopeful |
| **Pain Points** | Time wasters, lowball offers |
| **Opportunities** | Buyer reputation visible, offer management |

#### 6. Sale Completion

| Aspect | Details |
|--------|---------|
| **Touchpoint** | Messages, Mark as Sold dialog |
| **Actions** | Arranges handoff, receives payment, marks sold |
| **Thoughts** | "Success! Good experience" |
| **Emotion** | Delighted |
| **Opportunities** | Easy mark-as-sold flow, suggest re-listing |

#### 7. Feedback Loop

| Aspect | Details |
|--------|---------|
| **Touchpoint** | Rating system |
| **Actions** | Rates buyer, receives rating |
| **Thoughts** | "Building my reputation" |
| **Emotion** | Satisfied |
| **Opportunities** | Highlight seller achievements, badges |

## Journey 3: Forum Question Asker

**Persona:** Any user with a bonsai question
**Goal:** Get expert advice on bonsai care issue

### Stages

#### 1. Problem Encounter

| Aspect | Details |
|--------|---------|
| **Trigger** | Tree showing problems, unsure of care step |
| **Thoughts** | "Something's wrong, I need help fast" |
| **Emotion** | Worried |

#### 2. Search for Answers

| Aspect | Details |
|--------|---------|
| **Touchpoint** | Forum, Google (SEO) |
| **Actions** | Searches existing posts, browses categories |
| **Thoughts** | "Has someone asked this before?" |
| **Emotion** | Hopeful |
| **Pain Points** | Can't find relevant post |
| **Opportunities** | Better search, suggested similar posts |

#### 3. Ask Question

| Aspect | Details |
|--------|---------|
| **Touchpoint** | Forum Ask page |
| **Actions** | Writes question, uploads problem photos |
| **Thoughts** | "I hope someone can help" |
| **Emotion** | Hopeful, Vulnerable |
| **Pain Points** | Unsure which category |
| **Opportunities** | Auto-suggest category, photo upload prompts |

#### 4. Wait for Response

| Aspect | Details |
|--------|---------|
| **Touchpoint** | Notifications |
| **Actions** | Checks for responses |
| **Thoughts** | "Is anyone going to help?" |
| **Emotion** | Anxious |
| **Pain Points** | Slow response time |
| **Opportunities** | Expert notification, gamification for responders |

#### 5. Receive Help

| Aspect | Details |
|--------|---------|
| **Touchpoint** | Forum post comments |
| **Actions** | Reads advice, asks follow-ups |
| **Thoughts** | "This community is helpful!" |
| **Emotion** | Relieved, Grateful |
| **Opportunities** | Mark best answer, thank responders |

#### 6. Resolution

| Aspect | Details |
|--------|---------|
| **Touchpoint** | Forum post |
| **Actions** | Marks question answered, upvotes helpful responses |
| **Thoughts** | "Problem solved, I learned something" |
| **Emotion** | Satisfied |
| **Opportunities** | Share success story, suggest marketplace |
